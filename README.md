# Expense Tracker App

A simple and clean **Expense Tracker** web application built with **Vanilla JavaScript**, HTML, and CSS.  
The app allows users to manage daily expenses with persistent data storage using **LocalStorage**.

This project focuses on **state-driven UI**, clean architecture, and real-world frontend logic.
